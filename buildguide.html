Mural 2.0 Build Quick Start

Prerequisite to Building Mural 2.0

The users must have the follow software installed. You can build Mural 2.0 on windows and Linux and Unix. This guide assumes that you do on the windows.

● Subversion client 1.7.2

● Apache Maven 2.0.9

● Apache Ant 1.8.0

● Oracle JDK 6.0

The users must have the follow software installed to test the build.

● Oracle NetBeans 7.0.1

● Oracle GlassFish 3.1.1

The users must have one of the supported databases installed.

● Databases support MySQL 5.1, Oracle 9i/10g/11g, Microsoft SQL Server 2005/2008.

Build Instructions

● Check out the mural 2.0 sources.

create the working folder C:\mural

svn checkout <URL>

● Edit environment variables

open and edit C:\mural\open-dm-mi\env.bat

● Setup environment variables

run C:\mural\open-dm-mi\env.bat

● Build the master index common jars by skipping the JUnit testing.

C:\mural\open-dm-mi\mvn clean install -Dmaven.test.skip=true

● Build the data integration common jars by skipping the JUnit testing.

C:\mural\open-dm-di\mvn clean install -Dmaven.test.skip=true

● Build the data quality common jars by skipping the JUnit testing.

C:\mural\open-dm-dq\mvn clean install -Dmaven.test.skip=true

● Build the master index netbeans modules jars by skipping the JUnit testing.

C:\mural\open-dm-mi\mvn clean install -Dmaven.test.skip=true

● Build the NetBeans plugin modules.

C:\mural\open-dm-di\ant -f build-mosaic make-mosaic


Build Deliverables

The Mural 2.0 Master Data Management NetBeans module is located under

● C:\mural\mosaic-dist\netgen-mosaic-2.0.nbm

The Mural 2.0 MultiDomain Relationship Management NetBeans module is located under

● C:\mural\mosaic-dist\netgen-mosaic-multidomain-2.0.nbm


Test Build

Refer to Quick Start Guide.
