Mural Master Data Management 2.0

 
 

Quick Start for Basic Features

Welcome to the Quick Start Guide. This guide describes basic steps for using Mural master data management 2.0.

Before using the procedures in this document you must have already installed the NetBeans software and the application server software.

Install Mural 2.0 Master Index Plug-in

1. Start NetBeans.

2. From the NetBeans Tools menu, click Plugins.

3. Click the Downloaded tab and then click Add Plugins.

4. Navigate to the location where you saved netgen-mosaic-2.0.nbm, select the file, and then click Open.

5. Click Install.

6. On the Installer, click Next, select the license acceptance check box, and then click Install again.

7. If a validation warning window appears, click Continue.

8. When the installation is complete, click Finish and close the Plugins window.

Install Mural 2.0 Multi-domain Relationship Management Plug-in

1. Start NetBeans.

2. From the NetBeans Tools menu, click Plugins.

3. Click the Downloaded tab and then click Add Plugins.

4. Navigate to the location where you saved netgen-mosaic-multidomain2.0.nbm, select the file, and then click Open.

5. Click Install.

6. On the Installer, click Next, select the license acceptance check box, and then click Install again.

7. If a validation warning window appears, click Continue.

8. When the installation is complete, click Finish and close the Plugins window.
